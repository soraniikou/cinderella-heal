    人には痛かった過去があるかもしれない
    私には思い出したくない過去がありましたが、当時の自分をhealingしたくなり、短いアプリ作りました

    1.扉をタップ
    2.痛みや忘れたい何かを記入→healingボタンを押す
    3.流れ星と短いメッセージ出てきます
    4.ラストメッセージ

    

[こちらから体験いただけます](https://cinderella-heal.vercel.app/)

　　

 無理しなくてだいじょうぶ　気がむいたらお使いください
